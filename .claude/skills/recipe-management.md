# RecipeAI 레시피 관리 스킬

## 개요

이 스킬은 RecipeAI 앱을 사용하여 음식 사진을 분석하고, 재료를 관리하며, 레시피를 단계별로 실행하고, 최적의 조미료 배합을 구성하는 완벽한 워크플로우를 제공합니다.

## 핵심 3단계

### 1단계: 사진 분석
**목표**: AI가 음식/음료를 정확하게 인식하고 기본 정보 추출

#### A. 음식별 최적 촬영 팁

**1) 일반 음식 & 요리**
```
촬영 조건:
- 자연광 또는 밝은 실내 조명 (태양광 직사 피하기)
- 45도 각도로 촬영 (입체감)
- 전체 음식이 프레임에 보이도록
- 그릇/접시도 함께 촬영 (음식 크기 파악)
- 깔끔한 배경 (흰색, 회색 권장)

예시: 밥, 국, 면, 반찬, 스테이크, 파스타, 피자 등
```

**2) 음료 (커피, 차, 주스 등)**
```
촬영 조건:
- 유리잔을 투명하게 보이도록 (역광 활용)
- 정면에서 촬영 (음료 색상 명확하게)
- 잔의 상단부 포함 (음료 상태 파악)
- 따뜻한 음료면 수증기도 함께 (생생함)
- 음료의 색상이 자연스럽게 보이도록

예시: 커피, 홍차, 녹차, 우롱차, 과일 주스, 우유 등
```

**3) 스무디 & 음료 합성**
```
촬영 조건:
- 밝은 조명 (투명함과 색상 살리기)
- 정면 또는 45도 (색층이 보이도록)
- 투명 잔 사용 시 배경 명확히
- 토핑 (과일, 우유거품 등) 포함
- 신선함을 표현하는 얼음, 민트 등 포함

예시: 딸기 스무디, 초콜릿 스무디, 그린 스무디 등
```

**4) 테이크아웃 음식**
```
촬영 조건:
- 포장에서 꺼내거나 포장 채로 촬영
- 용기 크기로 음식 크기 파악
- 소스, 반찬이 보이도록
- 신선한 상태에서 빨리 촬영

예시: 치킨, 피자, 국수, 샌드위치 등
```

**5) 디저트 & 베이커리**
```
촬영 조건:
- 수직(위에서 아래로) 또는 45도 각도
- 자연광 (따뜻한 빛)
- 크림, 토핑 디테일 보이도록
- 깔끔한 접시 또는 받침대
- 갓 만든 상태에서 촬영

예시: 케이크, 쿠키, 도넛, 아이스크림, 마카롱 등
```

**6) 한식 요리**
```
촬영 조건:
- 45도 각도 (밥, 국 함께 보이도록)
- 따뜻함 표현 (수증기 포함)
- 반찬 여러 개를 동시에 보여주기
- 전통 식기 또는 한식당 분위기

예시: 불고기, 비빔밥, 된장찌개, 김치, 구이 등
```

#### B. 갤러리에서 선택하는 경우
```
권장사항:
- 최신 또는 고화질 사진 선택
- 음식/음료가 선명하게 보이는 사진
- 너무 어둡거나 흐릿한 사진 피하기
- 한 가지 음식/음료만 촬영된 사진 (혼합 피하기)
- 자동 화이트밸런스가 된 사진 (색감 자연스럽게)

좋은 예: 유명 음식점 사진, 잡지사 음식 사진
피할 예: 집단 사진에 음식만 일부 포함, 야외 야간 촬영
```

#### C. AI 분석 결과 검증 (음식/음료별)

**음식류 검증**
```
분석 후 확인할 항목:
✓ 음식명이 정확한가?
✓ 인식된 재료가 맞는가?
✓ 빠진 재료는 없는가?
✓ 조리 난이도가 적절한가?
✓ 조리 시간이 현실적인가?
```

**음료류 검증**
```
분석 후 확인할 항목:
✓ 음료명이 정확한가?
✓ 기본 재료(물, 우유, 얼음 등) 인식했나?
✓ 향신료나 추가 재료 누락은 없나?
✓ 온도 조건(뜨거운/찬) 맞나?
✓ 만드는 시간이 현실적인가?
```

**스무디 검증**
```
분석 후 확인할 항목:
✓ 주요 과일/야채 인식했나?
✓ 우유 또는 대체 액체 인식했나?
✓ 토핑 (그래놀라, 견과류 등) 포함했나?
✓ 얼음 비율이 적절한가?
✓ 블렌딩 방법이 명확한가?
```

**분석 재시도 필요한 경우**:
- 음식/음료명이 틀렸음 (다른 각도에서 다시 촬영)
- 핵심 재료가 누락됨 (사진 품질 문제)
- 조리법이 일반적이지 않음 (특수 요리/음료)
- 색상이 왜곡됨 (조명 문제 - 다시 촬영)
- 음료 종류를 잘못 인식 (유사 음료와 혼동)

### 2단계: 재료 관리 및 준비

#### A. 재료 리스트 체계화
```
RecipeAI 앱의 재료 섹션:
├── 주요 재료 (단백질, 탄수화물)
├── 보조 재료 (야채, 버터 등)
├── 조미료 (소금, 후추, 간장 등)
└── 선택 재료 (더 나은 맛을 위한)
```

#### B. 재료 준비 체크리스트
```
요리 전 준비 (Mise en place):

1. 주요 재료 확인
   □ 수량 확인
   □ 신선도 확인 (유통기한, 색상, 냄새)
   □ 필요시 손질 (씻기, 자르기, 손질)

2. 보조 재료 준비
   □ 야채: 씻기 → 자르기 (크기 균일)
   □ 양념: 미리 섞기
   □ 액체: 측정하기

3. 조미료 준비
   □ 주요 조미료: 한곳에 배치
   □ 재료별 계량 도구 준비
   □ 나중에 넣을 조미료 표시

4. 도구 준비
   □ 냄비, 팬, 그릇 배치
   □ 주걱, 나무숟가락 준비
   □ 측정 도구 (스푼, 계량컵)
```

#### C. 재료 최적화 (저렴하고 쉽게 구하기)
```
기본 원칙:
- 제철 재료 사용 (가격 저렴, 신선도 좋음)
- 마트 저가 재료 (PB상품) 활용
- 대체 재료 활용 (비슷한 맛, 훨씬 저렴)

추천하는 저렴한 기본 재료:
- 달걀: 항상 쌈 (단백질 최고)
- 양파, 마늘, 당근: 제철마다 매우 쌈
- 감자, 무: 으뜸 저가 재료
- 계란, 두부: 일반 마트의 기본
- 일반 간장, 소금, 설탕: 대형마트 저가

대체 재료로 비용 절감:
- 올리브유 → 카놀라유 또는 포도씨유
- 참기름 → 카놀라유 + 소금 (풍미 유사)
- 생강 → 생강 가루 (더 저렴)
- 신선 허브 → 건조 허브 (저렴, 오래 보관)
- 프리미엄 소스 → 기본 간장 + 설탕
```

### 3단계: 레시피 단계별 실행

#### A. 단계 이해하기
```
각 단계의 의미:
1단계: 준비 (재료 손질, 양념)
2단계: 주요 조리 (볶기, 끓이기)
3단계: 맛 조절 (간 맞추기)
4단계: 완성 (플레이팅, 장식)
```

#### B. 조리 중 주의사항
```
타이밍 관리:
- 각 단계의 예상 시간 파악
- 동시에 진행할 수 있는 작업 파악
- 불 강도 미리 확인

맛 확인:
- 중간 중간 맛보기
- 너무 짜지 않은지 확인
- 부족한 맛이 있으면 기록
```

#### C. 문제 해결
```
일반적인 문제와 해결:

1. 맛이 너무 짠 경우
   → 물이나 국물 추가
   → 신맛(식초, 레몬) 추가로 균형잡기

2. 너무 매운 경우
   → 우유, 요구르트, 버터 추가
   → 당분(설탕, 꿀) 추가

3. 맛이 밋밋한 경우
   → 조미료 추가 (간장, 고추장)
   → 향신료 추가 (마늘, 생강, 고추)
   → 신맛 추가 (식초, 레몬)

4. 식감이 부드러운 경우
   → 신선한 채소 추가
   → 견과류 토핑 추가
   → 온도 재조절
```

### 4단계: 최적의 조미료 배합

#### A. 기본 조미료 배합 공식
```
RecipeAI의 "팁" 섹션에서 추천하는 배합:

기본 비율:
소금 : 설탕 : 식초 : 간장 = 1 : 0.5 : 1 : 0.5

조정 방법:
├── 한식: 간장 + 고추장 강화
├── 중식: 간장 + 생강 + 마늘
├── 양식: 올리브유 + 마늘 + 소금
└── 일식: 간장 + 미림 + 사케
```

#### B. 조미료별 역할

```
주요 조미료와 기능:

1. 소금 (기본)
   역할: 맛 강화, 방부
   사용량: 음식의 0.5~1%
   타이밍: 조리 초반 또는 완성 직전

2. 설탕 (조화)
   역할: 맛 부드럽게, 맛 통일
   사용량: 소금의 30~50%
   타이밍: 소금 다음

3. 식초/레몬 (신맛)
   역할: 신선함, 맛 도드라지게
   사용량: 소수 (10ml 정도)
   타이밍: 완성 직전

4. 간장 (감칠맛)
   역할: 깊이, umami 강화
   사용량: 1~2스푼
   타이밍: 중간 또는 마지막

5. 향신료 (마늘, 생강)
   역할: 특색있는 맛
   사용량: 재료의 1~5%
   타이밍: 초반 (향 우러내기)

6. 기름 (올리브유, 참기름)
   역할: 부드러움, 풍미
   사용량: 1~2스푼
   타이밍: 완성 직전
```

#### C. 조미료 배합 최적화 프로세스

```
Step 1: RecipeAI 기본 추천 검토
- 앱에서 제시한 조미료 배합 확인
- 자신의 입맛과 비교

Step 2: 시작 레벨 설정 (추천량의 80%)
- 예: 소금 1스푼 → 0.8스푼으로 시작
- 이유: 추가는 쉽지만 빼기는 어려움

Step 3: 단계별 맛보기
- 각 주요 단계마다 맛보기
- 부족한 맛 기록

Step 4: 조정 (5분 간격)
- 작은 양부터 추가 (1/4 스푼)
- 5분 기다렸다가 다시 맛보기
- 원하는 맛이 될 때까지 반복

Step 5: 최종 조정 (완성 직전)
- 신맛 (식초/레몬) 한두 방울
- 마지막 간 맞추기
- 맛 확정

Step 6: 기록
- RecipeAI 앱에 "팁" 섹션 수정
- 자신의 최적 배합 저장
- 다음에 같은 요리할 때 참고
```

#### D. 음식/음료별 배합 예시

**음식 요리 (저렴 옵션)**
```
1. 계란 요리 (계란프라이, 계란말이)
기본: 소금 + 후추
추천: + 식용유 + 마늘
최저가: 소금 1/4스푼 + 후추 약간 + 식용유 1스푼
구입처: 마트 PB상품 (계란, 소금, 후추, 식용유 저가)

2. 국/탕 요리
기본: 소금 + 간장
추천: + 마늘 + 무/당근
최저가: 소금 1/2스푼 + 간장 1스푼 + 마늘 1스푼
구입처: 큰마트 PB간장 (매우 저렴)

3. 볶음 요리 (가장 저렴)
기본: 간장 + 설탕 + 마늘
추천: + 양파 + 당근 + 식용유
최저가: 간장 1.5스푼 + 설탕 0.5스푼 + 마늘 1스푼 + 식용유 1스푼
★ 팁: 양파, 당근, 마늘은 대형마트에서 가장 저렴한 채소

4. 밥반찬 (초저가)
기본: 소금 + 간장 + 참기름 대체(식용유)
추천: + 계란 + 양파
최저가: 소금 적당량 + 간장 1스푼 + 식용유 1스푼 + 계란 1개
★ 팁: 계란과 양파만으로 맛있는 밥반찬 완성
```

**음료 (저렴 옵션)**
```
1. 커피 (집에서 저렴하게)
기본: 인스턴트 커피 + 물
추천: + 설탕 + 우유 (분말 우유도 가능)
최저가: 인스턴트 커피 1스푼 + 뜨거운 물 200ml + 설탕 1스푼
★ 팁: 원두 대신 인스턴트 커피 사용 (매우 저렴, 편함)
★ 우유 대신: 분말 우유 또는 일반 우유 (가장 저렴한 PB 우유)

2. 차 (가장 저렴한 음료)
기본: 찻주머니 또는 티백 + 물
추천: + 설탕 + 무가당
최저가: 일반 홍차 또는 보리차 티백 1개 + 끓는물 200ml + 설탕 필수 아님
★ 팁: 대형마트 PB 차 매우 저렴 (100개에 2,000원 정도)
★ 보리차/옥수수차: 더 저렴한 대체 옵션

3. 수정과/식혜 (전통 음료, 매우 저렴)
기본: 말린 생강 또는 당(설탕) + 물
추천: + 잣 또는 견과류
최저가: 설탕 2스푼 + 물 500ml + 생강가루 약간
★ 팁: 전통 음료는 재료가 매우 저렴함

4. 과일 주스 (계절 과일 활용)
기본: 제철 과일 + 물
추천: + 설탕 (과일이 시면 추가)
최저가: 제철 과일 1개 + 물 200ml + 설탕 0.5스푼
★ 팁: 여름 딸기, 수박 / 가을 포도, 배 (가장 저렴한 시즌)
★ 냉동과일 사용 (신선 과일보다 훨씬 쌈)
```

**스무디 (저렴하게 만들기)**
```
1. 바나나 스무디 (가장 저렴)
기본: 바나나 + 우유 + 얼음
추천: + 설탕 또는 꿀 (선택)
최저가: 바나나 1개 + 우유 200ml + 얼음 한줌
★ 팁: 바나나는 가장 저렴한 과일, 숙성된 바나나 활용
★ 우유 대신: 분말 우유 + 물 (더 저렴)

2. 냉동과일 스무디 (언제나 저가)
기본: 냉동 딸기/블루베리 + 우유 + 얼음
추천: + 설탕
최저가: 냉동과일 1컵 + 우유 200ml + 설탕 0.5스푼
★ 팁: 신선 과일보다 냉동과일이 40% 이상 저렴
★ 냉동 과일은 영양가 동일, 더 오래 보관 가능

3. 우유 스무디 (초저가)
기본: 우유 + 설탕 + 얼음
추천: + 바나나 또는 딸기 가루
최저가: 우유 300ml + 설탕 1스푼 + 얼음
★ 팁: 우유만으로도 맛있음, 과일 가루 추가로 맛 업그레이드

4. 야채 스무디 (건강함 + 저가)
기본: 당근 또는 양파(삶은 것) + 우유
추천: + 바나나 + 설탕
최저가: 당근 1개 + 우유 200ml + 바나나 1/2개 + 설탕 0.5스푼
★ 팁: 야채는 가장 저렴한 식재료
★ 당근 달콤함으로 설탕 최소화 가능

5. 콩 스무디 (영양 풍부, 저가)
기본: 검은콩 또는 팥(통조림) + 우유 + 설탕
추천: + 바나나
최저가: 통조림 콩 0.5컵 + 우유 200ml + 설탕 1.5스푼
★ 팁: 통조림 콩이 생콩보다 저렴하고 편함
★ 단백질 많아서 영양 가득
```

## 실전 워크플로우 (전체 과정)

### 초급 (처음 요리/음료를 만드는 경우)

**음식 요리**
```
1. 사진 촬영 또는 선택
   └─ 밝은 곳에서, 음식이 명확하게 보이도록

2. AI 분석 (RecipeAI 실행)
   └─ 분석 결과 확인

3. 재료 준비 (RecipeAI 체크리스트 확인)
   └─ 앱의 재료 리스트 그대로 준비
   └─ 배합은 앱의 추천을 따름

4. 단계별 조리 (RecipeAI 단계 확인)
   └─ 각 단계를 순차적으로 따름
   └─ 중간 맛보기 (필수)

5. 조미료 최종 조정
   └─ 앱의 팁 확인
   └─ 기본 배합에서 시작
   └─ 필요시 5분 간격으로 조정

6. 저장
   └─ RecipeAI 앱에 "저장하기"로 완성된 요리 저장
```

**음료 만들기 (커피, 차, 주스)**
```
1. 사진 촬영 또는 선택
   └─ 유리잔의 투명함이 보이도록
   └─ 음료의 색상이 자연스럽게

2. AI 분석 (RecipeAI 실행)
   └─ 음료명 확인
   └─ 기본 재료 확인

3. 재료 준비
   └─ 물, 우유, 얼음 등 기본 준비
   └─ 분량 확인

4. 단계별 제조
   └─ 온도 조절 (뜨거운/찬 음료)
   └─ 섞기, 우려내기 등 순서 따르기

5. 최종 조정
   └─ 단맛/신맛 조절
   └─ 온도 확인

6. 저장
   └─ RecipeAI에 저장
```

**스무디 만들기**
```
1. 사진 촬영
   └─ 색상이 잘 보이도록
   └─ 토핑까지 명확하게

2. AI 분석
   └─ 주요 과일/야채 확인
   └─ 필요한 액체 확인

3. 재료 준비
   └─ 과일/야채 씻기, 자르기
   └─ 우유/요구르트 준비
   └─ 얼음 준비

4. 블렌딩
   └─ 액체 먼저, 과일/야채 추가
   └─ 얼음 마지막에 추가
   └─ 원하는 농도까지 블렌딩

5. 토핑 추가
   └─ 그래놀라, 견과류 등
   └─ 신선한 과일 장식

6. 저장
   └─ RecipeAI에 저장
```

### 중급 (경험 있는 경우)

**음식 요리**
```
1. 사진 촬영 (최적 조건)
   └─ 45도 각도, 그릇 포함
   └─ 프로 사진처럼 구성

2. AI 분석 결과 검토
   └─ 빠진 재료 추가 확인
   └─ 자신의 스타일로 조정

3. 재료 최적화
   └─ 프리미엄 재료로 업그레이드
   └─ 지역산 또는 시즈널 재료 활용
   └─ 대체 재료 활용

4. 자신의 노하우 추가
   └─ 온도, 불 강도 조절
   └─ 추가 재료/향신료 고려
   └─ 조리 시간 단축 팁

5. 조미료 커스터마이징
   └─ 자신의 입맛에 맞게 조정
   └─ 창의적 배합 시도

6. 피드백 기록
   └─ RecipeAI의 팁 섹션에 자신의 조정 기록
   └─ 다음 요리 시 참고
```

**음료 제조 (전문성 추가)**
```
1. 사진 촬영 (전문 수준)
   └─ 스튜디오 같은 조명
   └─ 음료의 물리적 성질 표현 (투명함, 거품 등)

2. AI 분석 + 자신의 경험
   └─ 기본 레시피 참고
   └─ 시즌별 변형 고려 (아이스/핫)

3. 음료별 전문성
   └─ 커피: 원두 선택, 추출 방법 개선
   └─ 차: 우려내는 온도/시간 조절
   └─ 주스: 신선도 판단, 유산소/무산소 처리

4. 맛 프로파일 조정
   └─ 진한 맛 vs 가벼운 맛
   └─ 단맛 조절
   └─ 향신료 추가 (계피, 카다몬 등)

5. 기록
   └─ "좋은 커피는 75도에서 추출"
   └─ "이 차는 3분 30초가 최적"
```

**스무디 프리미엄화**
```
1. 사진 (인스타그래머블)
   └─ 색상 조화
   └─ 토핑의 미적 배열

2. 재료 업그레이드
   └─ 오가닉 과일/야채
   └─ 프로틴 파우더, 슈퍼푸드 추가
   └─ 프리미엄 우유 선택 (오트, 아몬드 등)

3. 영양 최적화
   └─ 비타민 함량 고려
   └─ 단백질 추가
   └─ 식이섬유 증대

4. 맛 조합 창의성
   └─ 색다른 과일 조합
   └─ 허브 추가 (민트, 바질 등)
```

### 고급 (마스터 수준)

**마스터 요리사**
```
1. 사진 분석 (AI 제안 참고용)
   └─ 기본 가이드로만 사용
   └─ 대부분 자신의 경험에 의존

2. 재료 선택 및 조합
   └─ 품질 중심 (원산지, 시즌, 신선도)
   └─ 창의적 대체/추가
   └─ 미슐랭 수준의 프레젠테이션

3. 조미료 음악 (Flavor Symphony)
   └─ 기본 비율을 바탕으로 예술적 조정
   └─ 예: "이 요리는 약간의 고추장이 필요해"
   └─ 각 조미료의 타이밍 완벽 조절
   └─ 미세한 조정으로 완벽한 맛 구현

4. RecipeAI 활용
   └─ 새로운 요리 아이디어 찾기
   └─ 자신의 노하우 기록 및 전승
   └─ 완성된 레시피 저장 및 공유
```

**마스터 바리스타 (음료 전문가)**
```
1. 원두/차엽 선택
   └─ 원산지별 특성 이해
   └─ 로스트 레벨에 따른 맛 차이
   └─ 블렌딩으로 시그니처 음료 창작

2. 추출/우려내기 완벽화
   └─ 온도, 시간을 0.1도, 1초 단위로 조절
   └─ 물의 경도 고려
   └─ 기계 또는 수동 추출의 장단점 활용

3. 라떼 아트 & 프레젠테이션
   └─ 거품 기술 완벽화
   └─ 미적 표현
   └─ 온도 관리

4. 음료 큐레이션
   └─ 계절 음료 개발
   └─ 고객 입맛에 맞는 맞춤 음료
   └─ RecipeAI로 새로운 조합 발견
```

**스무디 아티스트**

   └─ 각 과일/야채의 영양가 완벽 이해
   └─ 영양 불균형 해소 조합

3. 비주얼 퍼펙션

## 조미료 배합 마스터 팁

### 황금 법칙
```
1. 작게 시작하기
   → 추가는 쉽지만 빼기는 어려움
   → 항상 권장량의 70~80%에서 시작
   ★ 비용 절감: 재료를 쓸데없이 버리지 않기

2. 타이밍이 중요
   → 각 조미료의 최적 시간에 추가
   → 너무 일찍 → 향 손실
   → 너무 늦게 → 배합 못함
   ★ 비용 절감: 불필요한 재료 추가 방지

3. 맛의 균형 (기본 조미료로 충분)
   → 짠맛(소금) + 단맛(설탕) + 신맛(식초) + 향신료(마늘)
   → 이 4가지만으로 대부분의 맛 표현 가능
   ★ 비용 절감: 비싼 조미료 불필요

4. 질감 고려 (저가 재료로 표현)
   → 부드러운 음식 → 신맛(식초) 추가 (입맛 자극)
   → 진한 음식 → 단맛(설탕) 추가 (조화)
   ★ 비용 절감: 간단한 재료로 식감 개선

5. 저가 기본 재료로 충분
   → 소금, 설탕, 간장, 식초 (기본 4종)
   → 마늘, 양파 (향신료 대체)
   → 계란, 우유 (다재다능)
   ★ 팁: 비싼 프리미엄 재료 없어도 맛있음
```

### 재사용 가능한 조미료 세트 (초저가)
```
모든 요리에 쓸 수 있는 기본 세트 (총 비용 3,000원 이하):
★ 필수 5가지 (이것만 있어도 99% 요리 가능)
- 소금 (일반 소금, 매우 저가)
- 설탕 (백설탕, 최저가)
- 간장 (PB 간장, 저렴)
- 식용유 (일반 카놀라유, 최저가)
- 마늘 (신선 마늘, 항상 저렴)

★ 추가 5가지 (있으면 더 좋음, 모두 저렴)
- 후추 (분말 후추, 저가)
- 식초 (양조 식초, 저렴)
- 양파 (항상 저렴)
- 당근 (항상 저렴)
- 계란 (가장 저렴한 단백질)

비용 절감 팁:
→ 올리브유 대신 카놀라유/포도씨유 (50% 저렴)
→ 참기름 대신 식용유 + 소금 (90% 저렴)
→ 신선 생강 대신 생강가루 (더 저렴, 오래 보관)
→ 신선 허브 대신 건조 향신료 (매우 저렴)
→ 프리미엄 간장 대신 일반 PB간장 (70% 저렴)

이 기본 세트만으로도 셰프 수준의 요리 가능!
```

### 조미료 저장 가이드
```
최적 보관:
- 상온 (어두운 곳): 향신료, 기름
- 냉장: 신선한 마늘, 생강
- 냉동: 고급 향신료

사용 기한:
- 분말 향신료: 1년
- 액체 (식초, 간장): 2년 이상
- 기름: 6개월 (개봉 후)
```

## RecipeAI 앱과의 통합

### 앱 활용 팁
```
1. "팁" 섹션 활용
   → 자신의 조미료 배합 기록
   → "소금 1/4스푼에서 시작, 최종 1/2스푼이 최적"

2. 이미지 저장
   → 최종 완성된 요리 사진 저장
   → 다음에 같은 음식 할 때 참고

3. 재료 체크리스트
   → 재료 리스트 출력 또는 사진
   → 마트 가기 전에 확인

4. 검색/필터
   → 저장된 요리 중 비슷한 것 찾기
   → 같은 재료 사용하는 요리 찾기
```

### 피드백 루프
```
매번 요리할 때마다:

1. 분석 (RecipeAI)
2. 실행 (자신의 방식)
3. 평가 (맛, 식감, 시간)
4. 기록 (앱의 팁 섹션 수정)
5. 다음 반복 (개선된 방식으로)

이 과정을 반복하면 마스터 요리사가 됨!
```

## 최종 체크리스트

요리할 때마다 이 순서대로 진행:

```
□ 사진 분석
  └─ 음식명 정확한가?
  └─ 재료 빠짐 없나?

□ 재료 준비
  └─ 모든 재료 준비됐나?
  └─ 신선도 확인했나?

□ 단계별 조리
  └─ 각 단계 이해했나?
  └─ 중간중간 맛봤나?

□ 조미료 최적화
  └─ 앱의 추천 확인했나?
  └─ 기본 70~80%에서 시작했나?
  └─ 5분 간격으로 조정했나?

□ 저장
  └─ 완성된 요리 저장했나?
  └─ 자신의 조미료 배합 기록했나?
```

이 스킬을 따르면 RecipeAI로 맛있고 일관된 요리가 가능합니다!
