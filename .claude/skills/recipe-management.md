# RecipeAI 레시피 관리 스킬

## 개요

이 스킬은 RecipeAI 앱을 사용하여 음식 사진을 분석하고, 재료를 관리하며, 레시피를 단계별로 실행하고, 최적의 조미료 배합을 구성하는 완벽한 워크플로우를 제공합니다.

## 핵심 3단계

### 1단계: 사진 분석
**목표**: AI가 음식을 정확하게 인식하고 기본 정보 추출

#### A. 사진 촬영 팁
```
최적의 사진 조건:
- 자연광 또는 밝은 실내 조명
- 음식을 45도 각도로 촬영 (입체감)
- 전체 음식이 프레임에 보이도록
- 그릇/접시도 함께 촬영 (음식 크기 파악)
- 깔끔한 배경 (흰색, 회색 권장)
```

#### B. 갤러리에서 선택하는 경우
```
권장사항:
- 최신 또는 고화질 사진 선택
- 음식이 선명하게 보이는 사진
- 너무 어둡거나 흐릿한 사진 피하기
- 한 가지 음식만 촬영된 사진
```

#### C. AI 분석 결과 확인
```
분석 후 확인할 항목:
✓ 음식명이 정확한가?
✓ 인식된 재료가 맞는가?
✓ 빠진 재료는 없는가?
✓ 조리 난이도가 적절한가?
```

**분석 재시도 필요한 경우**:
- 음식명이 틀렸음 (다른 각도에서 다시 촬영)
- 핵심 재료가 누락됨 (사진 품질 문제)
- 조리법이 일반적이지 않음 (특수 요리)

---

### 2단계: 재료 관리 및 준비

#### A. 재료 리스트 체계화
```
RecipeAI 앱의 재료 섹션:
├── 주요 재료 (단백질, 탄수화물)
├── 보조 재료 (야채, 버터 등)
├── 조미료 (소금, 후추, 간장 등)
└── 선택 재료 (더 나은 맛을 위한)
```

#### B. 재료 준비 체크리스트
```
요리 전 준비 (Mise en place):

1. 주요 재료 확인
   □ 수량 확인
   □ 신선도 확인 (유통기한, 색상, 냄새)
   □ 필요시 손질 (씻기, 자르기, 손질)

2. 보조 재료 준비
   □ 야채: 씻기 → 자르기 (크기 균일)
   □ 양념: 미리 섞기
   □ 액체: 측정하기

3. 조미료 준비
   □ 주요 조미료: 한곳에 배치
   □ 재료별 계량 도구 준비
   □ 나중에 넣을 조미료 표시

4. 도구 준비
   □ 냄비, 팬, 그릇 배치
   □ 주걱, 나무숟가락 준비
   □ 측정 도구 (스푼, 계량컵)
```

#### C. 재료 최적화
```
비용 절감:
- 제철 재료 사용 (가격 저렴, 신선도 좋음)
- 대량 구매 시 보관 방법 고려
- 대체 재료 활용 (비슷한 맛, 저렴함)

품질 향상:
- 프리미엄 조미료 (간장, 참기름)
- 신선한 향신료 (마늘, 생강)
- 계절 식재료 우선 사용
```

### 3단계: 레시피 단계별 실행

#### A. 단계 이해하기
```
각 단계의 의미:
1단계: 준비 (재료 손질, 양념)
2단계: 주요 조리 (볶기, 끓이기)
3단계: 맛 조절 (간 맞추기)
4단계: 완성 (플레이팅, 장식)
```

#### B. 조리 중 주의사항
```
타이밍 관리:
- 각 단계의 예상 시간 파악
- 동시에 진행할 수 있는 작업 파악
- 불 강도 미리 확인

맛 확인:
- 중간 중간 맛보기
- 너무 짜지 않은지 확인
- 부족한 맛이 있으면 기록
```

#### C. 문제 해결
```
일반적인 문제와 해결:

1. 맛이 너무 짠 경우
   → 물이나 국물 추가
   → 신맛(식초, 레몬) 추가로 균형잡기

2. 너무 매운 경우
   → 우유, 요구르트, 버터 추가
   → 당분(설탕, 꿀) 추가

3. 맛이 밋밋한 경우
   → 조미료 추가 (간장, 고추장)
   → 향신료 추가 (마늘, 생강, 고추)
   → 신맛 추가 (식초, 레몬)

4. 식감이 부드러운 경우
   → 신선한 채소 추가
   → 견과류 토핑 추가
   → 온도 재조절
```

### 4단계: 최적의 조미료 배합

#### A. 기본 조미료 배합 공식
```
RecipeAI의 "팁" 섹션에서 추천하는 배합:

기본 비율:
소금 : 설탕 : 식초 : 간장 = 1 : 0.5 : 1 : 0.5

조정 방법:
├── 한식: 간장 + 고추장 강화
├── 중식: 간장 + 생강 + 마늘
├── 양식: 올리브유 + 마늘 + 소금
└── 일식: 간장 + 미림 + 사케
```

#### B. 조미료별 역할

```
주요 조미료와 기능:

1. 소금 (기본)
   역할: 맛 강화, 방부
   사용량: 음식의 0.5~1%
   타이밍: 조리 초반 또는 완성 직전

2. 설탕 (조화)
   역할: 맛 부드럽게, 맛 통일
   사용량: 소금의 30~50%
   타이밍: 소금 다음

3. 식초/레몬 (신맛)
   역할: 신선함, 맛 도드라지게
   사용량: 소수 (10ml 정도)
   타이밍: 완성 직전

4. 간장 (감칠맛)
   역할: 깊이, umami 강화
   사용량: 1~2스푼
   타이밍: 중간 또는 마지막

5. 향신료 (마늘, 생강)
   역할: 특색있는 맛
   사용량: 재료의 1~5%
   타이밍: 초반 (향 우러내기)

6. 기름 (올리브유, 참기름)
   역할: 부드러움, 풍미
   사용량: 1~2스푼
   타이밍: 완성 직전
```

#### C. 조미료 배합 최적화 프로세스

```
Step 1: RecipeAI 기본 추천 검토
- 앱에서 제시한 조미료 배합 확인
- 자신의 입맛과 비교

Step 2: 시작 레벨 설정 (추천량의 80%)
- 예: 소금 1스푼 → 0.8스푼으로 시작
- 이유: 추가는 쉽지만 빼기는 어려움

Step 3: 단계별 맛보기
- 각 주요 단계마다 맛보기
- 부족한 맛 기록

Step 4: 조정 (5분 간격)
- 작은 양부터 추가 (1/4 스푼)
- 5분 기다렸다가 다시 맛보기
- 원하는 맛이 될 때까지 반복

Step 5: 최종 조정 (완성 직전)
- 신맛 (식초/레몬) 한두 방울
- 마지막 간 맞추기
- 맛 확정

Step 6: 기록
- RecipeAI 앱에 "팁" 섹션 수정
- 자신의 최적 배합 저장
- 다음에 같은 요리할 때 참고
```

#### D. 음식별 조미료 배합 예시

```
1. 계란 요리 (계란프라이, 계란말이)
기본: 소금 + 후추
심화: + 버터 + 마늘
최적: 소금 1/4스푼 + 후추 약간 + 버터 1/2스푼

2. 국/탕 요리
기본: 소금 + 간장
심화: + 마늘 + 생강 + 파
최적: 소금 1/2스푼 + 간장 1스푼 + 마늘 다진 것 1스푼

3. 볶음 요리
기본: 간장 + 설탕 + 마늘
심화: + 생강 + 고추 + 참기름
최적: 간장 1.5스푼 + 설탕 0.5스푼 + 마늘 1스푼 + 참기름 0.5스푼

4. 샐러드
기본: 올리브유 + 식초 + 소금
심화: + 마늘 + 겨자 + 꿀
최적: 올리브유 3스푼 + 식초 1스푼 + 소금 약간 + 꿀 1/2스푼
```

## 실전 워크플로우 (전체 과정)

### 초급 (처음 요리하는 경우)
```
1. 사진 촬영 또는 선택
   └─ 밝은 곳에서, 음식이 명확하게 보이도록

2. AI 분석 (RecipeAI 실행)
   └─ 분석 결과 확인

3. 재료 준비 (RecipeAI 체크리스트 확인)
   └─ 앱의 재료 리스트 그대로 준비
   └─ 배합은 앱의 추천을 따름

4. 단계별 조리 (RecipeAI 단계 확인)
   └─ 각 단계를 순차적으로 따름
   └─ 중간 맛보기 (필수)

5. 조미료 최종 조정
   └─ 앱의 팁 확인
   └─ 기본 배합에서 시작
   └─ 필요시 5분 간격으로 조정

6. 저장
   └─ RecipeAI 앱에 "저장하기"로 완성된 요리 저장
```

### 중급 (경험 있는 경우)
```
1. 사진 촬영 (최적 조건)
   └─ 45도 각도, 그릇 포함

2. AI 분석 결과 검토
   └─ 빠진 재료 추가 확인
   └─ 자신의 스타일로 조정

3. 재료 최적화
   └─ 프리미엄 재료로 업그레이드
   └─ 대체 재료 활용

4. 자신의 노하우 추가
   └─ 온도, 불 강도 조절
   └─ 추가 재료/향신료 고려

5. 조미료 커스터마이징
   └─ 자신의 입맛에 맞게 조정
   └─ 창의적 배합 시도

6. 피드백 기록
   └─ RecipeAI의 팁 섹션에 자신의 조정 기록
   └─ 다음 요리 시 참고
```

### 고급 (마스터 요리사)
```
1. 사진 분석 (AI 제안 참고용)
   └─ 기본 가이드로만 사용
   └─ 대부분 자신의 경험에 의존

2. 재료 선택 및 조합
   └─ 품질 중심
   └─ 창의적 대체/추가

3. 조미료 음악 (Flavor Symphony)
   └─ 기본 비율을 바탕으로 예술적 조정
   └─ 예: "이 요리는 약간의 고추장이 필요해"
   └─ 각 조미료의 타이밍 완벽 조절

4. RecipeAI 활용
   └─ 새로운 요리 아이디어 찾기
   └─ 자신의 노하우 기록
   └─ 완성된 레시피 저장 및 공유
```

## 조미료 배합 마스터 팁

### 황금 법칙
```
1. 작게 시작하기
   → 추가는 쉽지만 빼기는 어려움
   → 항상 권장량의 70~80%에서 시작

2. 타이밍이 중요
   → 각 조미료의 최적 시간에 추가
   → 너무 일찍 → 향 손실
   → 너무 늦게 → 배합 못함

3. 맛의 균형
   → 짠맛 + 단맛 + 신맛 + 향신료
   → 어느 하나도 지배적이지 않아야 함

4. 질감 고려
   → 부드러운 음식 → 신맛 추가 (입맛 자극)
   → 진한 음식 → 단맛 추가 (조화)
```

### 재사용 가능한 조미료 세트
```
모든 요리에 쓸 수 있는 기본 세트:
- 소금 (바다소금 권장)
- 설탕 (백설탕 또는 꿀)
- 검은 후추 (간 것)
- 마늘 (생것 또는 다진 것)
- 올리브유
- 식초 (백식초 또는 발사믹)

이 6가지만으로도 대부분의 요리 가능!
```

### 조미료 저장 가이드
```
최적 보관:
- 상온 (어두운 곳): 향신료, 기름
- 냉장: 신선한 마늘, 생강
- 냉동: 고급 향신료

사용 기한:
- 분말 향신료: 1년
- 액체 (식초, 간장): 2년 이상
- 기름: 6개월 (개봉 후)
```

## RecipeAI 앱과의 통합

### 앱 활용 팁
```
1. "팁" 섹션 활용
   → 자신의 조미료 배합 기록
   → "소금 1/4스푼에서 시작, 최종 1/2스푼이 최적"

2. 이미지 저장
   → 최종 완성된 요리 사진 저장
   → 다음에 같은 음식 할 때 참고

3. 재료 체크리스트
   → 재료 리스트 출력 또는 사진
   → 마트 가기 전에 확인

4. 검색/필터
   → 저장된 요리 중 비슷한 것 찾기
   → 같은 재료 사용하는 요리 찾기
```

### 피드백 루프
```
매번 요리할 때마다:

1. 분석 (RecipeAI)
2. 실행 (자신의 방식)
3. 평가 (맛, 식감, 시간)
4. 기록 (앱의 팁 섹션 수정)
5. 다음 반복 (개선된 방식으로)

이 과정을 반복하면 마스터 요리사가 됨!
```

## 최종 체크리스트

요리할 때마다 이 순서대로 진행:

```
□ 사진 분석
  └─ 음식명 정확한가?
  └─ 재료 빠짐 없나?

□ 재료 준비
  └─ 모든 재료 준비됐나?
  └─ 신선도 확인했나?

□ 단계별 조리
  └─ 각 단계 이해했나?
  └─ 중간중간 맛봤나?

□ 조미료 최적화
  └─ 앱의 추천 확인했나?
  └─ 기본 70~80%에서 시작했나?
  └─ 5분 간격으로 조정했나?

□ 저장
  └─ 완성된 요리 저장했나?
  └─ 자신의 조미료 배합 기록했나?
```

이 스킬을 따르면 RecipeAI로 맛있고 일관된 요리가 가능합니다!
